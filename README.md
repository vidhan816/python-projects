# Python Projects

This repository contains two Python-based projects:

---

## 📌 1. Expense Tracker with Data Visualization
- Tracks daily expenses
- Stores data in CSV file
- Uses Pandas for data analysis
- Uses Matplotlib for visualization (graphs)

### Features:
- Add expenses
- View category-wise summary
- Visualize monthly and category spending

---

## 📌 2. Employee Management System (Tkinter + MySQL)
- Desktop GUI application using Tkinter
- MySQL database integration
- Performs CRUD operations (Create, Read, Update, Delete)
- Includes login authentication system

### Features:
- Add, update, delete employee records
- Search employee by name
- Interactive GUI interface

---

## 🛠️ Technologies Used
- Python
- Pandas
- Matplotlib
- Tkinter
- MySQL

---

## ▶️ How to Run

### Expense Tracker:
```bash
python expense_tracker.py
```

### Employee Management System:
```bash
python employee_db.py
```
